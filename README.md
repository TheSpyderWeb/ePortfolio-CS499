## Artifacts

### Enhancement 1: Software Design and Engineering
- **Original Artifact:** Recipe Roulette basic web app  
- **Enhancement:** Improved UI/UX, added modular JavaScript architecture, implemented responsive design.  
- **Skills Demonstrated:** Frontend design, modular coding practices, user-centered design principles.  
- **Reflection:** I learned how to structure frontend code for maintainability and scalability. This enhancement meets course outcomes related to software design and engineering.  

### Enhancement 2: Algorithms and Data Structures
- **Original Artifact:** Recipe fetching and filtering logic  
- **Enhancement:** Optimized recipe search and filtering using efficient array and object operations; implemented caching for performance.  
- **Skills Demonstrated:** Data structure selection, algorithm optimization, computational efficiency.  
- **Reflection:** I gained experience in analyzing algorithm performance and improving responsiveness of web apps. This addresses course outcomes in algorithms and problem-solving.  

### Enhancement 3: Databases
- **Original Artifact:** Firebase integration for saving favorites  
- **Enhancement:** Expanded database schema to track user preferences, integrated Firestore queries with frontend, and added input validation.  
- **Skills Demonstrated:** Database design, query optimization, full-stack integration, data validation.  
- **Reflection:** This enhanced my understanding of how databases interact with web applications and improves user experience. Meets course outcomes related to database management and security.
